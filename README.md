# Técnicas de Programación

Este repositorio contiene los ejemplos desarrollados para la materia **Técnicas de Programación**.  
Su propósito es aplicar y demostrar las principales **técnicas de la Programación Orientada a Objetos (POO)**, utilizando el lenguaje **Python**.

## Contenido del repositorio

Cada carpeta representa una de las técnicas vistas en clase:

**Abstracción**  
  Ejemplo práctico que muestra cómo ocultar los detalles de implementación y trabajar con conceptos generales.  
  Archivo: `ejemplo_abstraccion.py`

**Encapsulación**  
  Ejemplo que ilustra cómo proteger los datos internos de una clase mediante el uso de métodos controladores.  
  Archivo: `ejemplo_encapsulacion.py`

**Herencia**  
  Demostración del uso de clases base y derivadas, permitiendo la reutilización de código.  
  Archivo: `ejemplo_herencia.py`

**Polimorfismo**  
  Ejemplo que evidencia cómo diferentes clases pueden compartir un mismo método, pero con comportamientos distintos.  
  Archivo: `ejemplo_polimorfismo.py`

## Objetivo general

Aplicar los conceptos de la programación orientada a objetos para desarrollar ejemplos que muestren de manera clara el funcionamiento de las técnicas de **abstracción**, **encapsulación**, **herencia** y **polimorfismo**.
